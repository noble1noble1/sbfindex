# sbfindex.com

Live counterfactual tracker of what Sam Bankman-Fried's pre-collapse FTX portfolio would be worth today.

Built by [@noble1noble1](https://x.com/noble1noble1).

## Stack
Next.js 16.2 (Turbopack), TypeScript, Tailwind 4. Static export. Deployed on Vercel.

## Data sources
- Forbes (Josipa Majic Predin, Mar 18, 2026)
- Anthropic press / CNBC IPO filing (Jun 1, 2026)
- SEC EDGAR (SpaceX S-1, May 20, 2026)
- CoinGecko (live SOL price)
- Yahoo Finance (HOOD)
- Primary FTX bankruptcy estate filings

## Local dev
```
npm install
npm run dev
```
